# ReadWise
A simple self hosted online library.
---

The system has the following user types:-
 - Admin
 - Users
 - Super User

## Admin User
This user has the following capabilities:-
1. Add/Update/Remove/List new PDF books. (Cover image, summary, PDF file, category and so on)

## Regular User
1. Browse the collection of existing books
   - Apply filters such as genre, author e.t.c

2. Recommendation System for similar books:-
   - Show similar books when the view a specific books.
   - Ask for AI recommendation based on their preferences.

3. Manage personal reading shelves e.g Technical Books e.t.c

4. Read a book using a PDF viewer. Add notes to the book they are reading.

5. Review + Rate a book
